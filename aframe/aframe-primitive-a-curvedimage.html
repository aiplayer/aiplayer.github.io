# AFrame基础控件-a-curvedimage

弯曲的图像基元创建围绕用户弯曲的图像。由于每个像素与用户之间的距离相同，因此围绕摄像机布置的弯曲图像可能更易于辨认。对于复杂的布局，与倾斜的平面相比，它们是更好的选择，因为它们可以确保光滑的表面，而不是平面之间的一系列尴尬的接缝。

在引擎盖下，弯曲的图像是双面开放式圆柱体，纹理映射到圆柱体内部。

# 例

<a-scene>
  <a-assets>
    <img id="my-image" src="image.png">
  </a-assets>

  <!-- Using the asset management system. -->
  <a-curvedimage src="#my-image" height="3.0" radius="5.7" theta-length="72"
                 rotation="0 100 0" scale="0.8 0.8 0.8"></a-curvedimage>

  <!-- Defining the URL inline. Not recommended but more comfortable for web developers. -->
  <a-curvedimage src="another-image.png"></a-curvedimage>
</a-scene>

# 微调

确保图像不被拉伸歪曲要求我们仔细设置的height，radius和theta-length相对于所述图像的纵横比属性。

根据这个公式：

S = r *θ

我们需要计算r或θ用于S向匹配height保留图像的原始长宽比值。在此公式中，θ以弧度设置，radius而S属性以米设置。

圆弧公式

以度为单位，thetaStart定义弧的起始thetaLength位置和弧的终止位置。

一旦对这些值进行了微调以避免失真，我们就可以 scale用来安全地调整曲面图像相对于用户的距离。
