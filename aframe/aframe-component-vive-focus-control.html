# Vive焦点组件

vive-focus-controls组件与Vive Focus控制器对接。它包装了跟踪控件组件，同时添加了按钮映射，事件以及突出显示触摸和/或按下按钮（触控板，触发器）的Vive Focus控制器模型。

# 例

<!-- Match Vive Focus controller if present, regardless of hand. -->
<a-entity vive-focus-controls></a-entity>

<!-- Match Vive Focus controller if present and for specified hand. -->
<a-entity vive-focus-controls="hand: left"></a-entity>
<a-entity vive-focus-controls="hand: right"></a-entity>

# 值

属性 描述 默认
armModel 手臂模型是否用于位置数据。 真正
buttonTouchedColor 触摸时按钮的颜色（仅触控板）。 ＃777777
buttonHighlightColor 按下并激活时按钮的颜色。 #FFFFFF
手 将被跟踪的手（例如，右，左）。
模型 是否已加载Vive Focus控制器模型。 真正
orientationOffset 偏移量应用于模型方向。 x：0，y：0，z：0

# 事件

活动名称 描述
触控板已更改 触控板已更改。
触控板 按下触控板。
触控板已移动 触控板轴已更改。
触控板 触控板发布。
触控板触摸开始 触控板感动了。
触控板触摸 触控板未触碰。
触发改变 触发器已更改。
触发 扣动扳机。
触发 触发器释放。

由于此控制器的按钮是数字按钮，因此更改的事件仅在完全按下或释放按钮（值0或1）时触发。
