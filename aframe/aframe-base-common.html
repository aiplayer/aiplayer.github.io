# VR 硬件与浏览器

# 什么是VR？

虚拟现实（Virtual reality，VR）是一种技术，它使用带有显示器的头戴式耳机来产生逼真的图像、声音和其他感觉，将用户置于沉浸式虚拟环境中。虚拟现实让我们创造出无限的世界，人们可以用手四处走动和互动，感觉自己仿佛被转移到了另一个地方。

# 耳机之间有什么区别？

市场上有几种具有不同功能的消费者虚拟现实耳机。重要的区别特征包括：

  • 具有位置跟踪（六自由度（六自由度））或仅具有旋转跟踪（三自由度（三自由度））。

  • 是否有控制器，以及这些控制器是否有6自由度或3自由度。一般来说，控制器的自由度与耳机的自由度相匹配。

  • 由个人电脑、移动设备或独立设备供电。

旋转跟踪允许人们环视或旋转物体。所有耳机提供旋转跟踪。

位置跟踪允许人们四处移动，接近物体，向前伸展。随着虚拟现实产业的发展，最起码的可行体验将趋向于拥有带有位置跟踪控制器的位置跟踪耳机。位置跟踪对于给人以存在感，让他们感觉自己身处真实的环境中很重要。由于只进行旋转跟踪，人们只能环顾四周并晃动控制器。

# 什么是WebVR？

WebVR是一个JavaScript API，用于在浏览器中创建沉浸式3D虚拟现实体验。或者简单地说，允许虚拟现实在浏览器中通过网络。

A-Frame使用WebVR API访问VR耳机传感器数据（位置、方向）来转换相机，并将内容直接呈现到VR耳机。请注意，提供数据的WebVR不应与提供图形和渲染的WebGL混淆或混淆。

# 哪些浏览器支持虚拟现实？

目前市场上支持VR的浏览器崩溃Supermedium和Exokit

# A-Frame想把WebVR带到哪里？

A-Frame旨在提供高度沉浸感和交互式的虚拟现实内容，并具有类似本地的性能。为此，A-Frame认为，最小可行棒将趋向于带有位置跟踪控制器的位置跟踪耳机。这是A-Frame想要创新和发现虚拟现实网络特有的新基础（例如，链接遍历、分散化、身份）的范例。将这类内容与平面和静态360°内容和菜单进行对比。

同时，A-Frame希望每个人都能参与到虚拟现实内容的创作中来。A-Frame支持所有主要耳机及其控制器。幸运的是，A-Frame拥有庞大的社区和贡献者，既能展望未来，又能满足当今虚拟现实景观的需求。

# A-Frame支持哪些平台？

A-Frame通过浏览器几乎支持所有平台。A型架支撑的一般平台包括：

  • 带耳机的桌面虚拟现实

  • 带耳机的移动虚拟现实

  • 独立耳机上的虚拟现实

  • 桌面平面（即鼠标和键盘）

  • 平板手机（即魔术窗口）

已证明可与A形架配合使用的其他一些平台包括：

  • AR耳机上的增强现实（AR）（如Magic Leap、HoloLens）

  • 移动增强现实（AR）（即magic window、ARKit、ARCore）

# A-Frame支持哪些虚拟现实耳机？

A-Frame通过浏览器支持大多数耳机。A-Frame支持的一些虚拟现实耳机包括：

  • HTC Vive
  • Oculus Rift
  • Oculus Quest
  • Oculus Go
  • Google Daydream
  • Samsung GearVR
  • Vive Focus
