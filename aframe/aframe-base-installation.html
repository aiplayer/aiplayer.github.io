# AFrame安装

本安装部分提供了几种开始使用A-Frame的方法，尽管大多数方法不需要任何实际安装，因为A-Frame主要是HTML和JavaScript。

# 浏览器中的代码编辑器

最快的方法是从浏览器中开始播放。

# 使用Glitch编码

Glitch提供了一个在线代码编辑器，可以即时部署和托管网站。编辑器支持前端和后端代码以及多个文件和目录。小故障让我们可以重新混合（即复制）现有项目，并使它们成为我们自己的，立即宿主和部署更改，以供大家查看。

点击这个A-Frame项目中的Remix，在index.HTML中处理HTML，并查看你的网站在每次更改时的实时发布！例如，基本的A-Frame故障发布在aframe.Glitch.me上，但我们将提供您自己的自定义URL名称。

以下是一些其他的a-frame起步例子：

  • aframe-aincraft - 飞机例子.
  • aframe-gallery - 360° 图片展览.
  • aframe-registry - 各种组件例子.
  • aframe-vaporwave - 一幅怀旧的未来场景.
  • networked-aframe - 多用户.

# 本地开发

# 使用本地服务器

对于下面的选项，我们应该使用本地服务器开发项目，以便正确地提供文件。本地服务器的选项包括：

下载Mongoose应用程序并从与HTML文件相同的目录中打开它。

在终端中运行python-m SimpleHTTPServer（或者python-m http.server for python 3），与您的HTML文件位于同一目录下。

在终端上运行npm install-g live server&&live server，与HTML文件位于同一目录下。

一旦运行服务器，我们就可以在浏览器中使用服务器运行的本地URL和端口（例如http://localhost:8000）打开项目。尽量不要使用不提供域的file://协议打开项目；绝对和相对URL可能不起作用。

# 下载源码

github上的源码包包括：

  • 一个简单的HTML文件，链接到当前版本
  • 一个本地开发环境
  • 一个部署流程

# 直接使用预编译JS文件

可以使用JS引用，直接引用已经编译好的JS：

<head>
  <script src="https://aframe.io/releases/1.0.4/aframe.min.js"></script>
</head>

# 通过npm安装

npm install aframe

在应用程序中使用：

require('aframe')

如果使用npm安装，并使用angular前端技术，还可以使用如下的命令进行初始化一个场景模板：

npm install -g angle && angle initscene
