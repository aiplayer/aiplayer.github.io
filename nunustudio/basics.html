# 编辑器基础

# 编辑器用户界面布局

nunuStudio编辑器是开发nunu应用程序的地方，它提供了在3D世界中创建和编辑对象的工具。如果您曾经使用过其他类似的框架（统一性，playcanvas，godot等）或3D建模软件（blender，maya，cinema4d等），则下面的某些概念可能已经为您所熟悉。

对象以树形结构组织（在下面的红色区域中可见），每个对象都有子对象，这些子对象继承父对象的位置，比例和旋转。如果父对象被移动，则子对象也随之移动。

可以用鼠标左键在对象浏览器中选择对象，当选择一个对象时，对象面板（蓝色）中将填充有关该对象的数据。在对象面板中，可以手动更改对象的每个

窗口的左侧用于选择工具并将对象添加到场景，工具用于使用鼠标更改对象属性，以下工具可用

  • 选择工具，用鼠标左键选择对象，然后双击也可以选择对象（即使选择工具未激活）

  • 移动工具，用于移动对象

  • 调整大小工具，用于调整对象的大小

  • 旋转工具，用于旋转对象

在选择工具的下方，有一个对象添加区域，可以在其中选择对象并将对象添加到对象浏览器上当前选定的场景。将鼠标拖到每个选项上时，将出现一个新框，其中显示了每个可用类别的对象。

资源浏览器区域（绿色），用于管理所有可用资源（图像，视频，字体等），将新资源导入项目（单击导入菜单，或者简单地将其拖动到资源浏览器中），甚至将资源从项目导出到主机系统。

# 在场景内导航

鼠标用于在场景编辑器中移动相机。有两种导航模式可用，默认情况下，“自由导航”模式已启用。

  • 要旋转相机，请按鼠标左键并四处移动鼠标（相机的移动类似于FPS游戏中相机的移动方式）。

  • 要移动相机，请按鼠标右键，然后移动鼠标

  • 要垂直移动，请按鼠标中键并上下移动鼠标

  • 要接近相机，请使用鼠标滚轮，如果使用触控板，请用手指捏合。

# 添加对象

要将您的第一个对象添加到场景中（以球体为例），请将鼠标移到左侧栏中的第5个图标，然后使用鼠标左键选择球体图标。

现在，使用选择工具选择添加的对象，然后选择移动工具并尝试将添加的对象移动到整个周围。

# 编辑物件

将对象添加到编辑器后，我们可以更改其属性，nunuStudio中的所有对象都有名称，位置，旋转度和比例。

要选择对象，可以选择右侧栏中的选择工具，然后在场景编辑器区域中单击该对象的顶部，或者也可以在对象浏览器中单击该对象。

当选择一个对象时，在左下角会显示一个可以更改的属性列表。不同类型的对象将具有不同的属性。

# 资源资源

对象可以附加有资源，资源可以是图像，视频，纹理，动画等。在屏幕底部，我们具有资源管理器，该资源管理器比可用于管理项目中的资源。

在资源浏览器上也有一个菜单，用于创建外部资源并将其导入到项目中。也可以在运行时从外部源导入资源，我们将在下一个教程的后面看到。

# 运行程序

要测试程序，请按顶部栏中的运行按钮，或按F5键，要停止该程序，请按停止按钮或再次按F5。
