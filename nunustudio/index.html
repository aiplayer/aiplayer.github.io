# 简介

nunustudio是一款基于threejs的3d场景编辑器。 本文介绍这个工具的基本使用，便于大家快速入门。

  • 编辑器基础：简单介绍一下编辑器UI和基本功能

  • 相机：介绍不同类型的相机及基本使用
  • 质材：解释什么是质材，及可用的质材
  • 粒子系统：介绍粒子系统及粒子编辑器
  • 声音：介绍声音效果，及脚本
  • 文本：介绍文本对象，创建方法，导入外部字体，控制文本对象
  • 物理：如何增加、使用和控制物理对象
  • 陀螺仪：使用陀螺仪控制对象，创建全景可视化
  • 虚拟现实：如何创建虚拟现实项目
  • 光线跟踪：使用光线投射器与3D对象通过鼠标互动
  • 动画：介绍如何创建或导入动画到编辑器中
  • 水：介绍如何使用质材创建3d水效果
  • 操纵杆：使用外部库集成一个操纵杆
  • 时间线：使用时间线编辑器创建动画
  • 后处理：后处理相机效果流水线
  • 视频：如何导入和使用视频文件
  • 着色器：使用GLSL着色器创建新材料
  • 地形：如何创建地形
